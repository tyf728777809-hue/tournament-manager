# 赛事管理系统数据表结构

## 1. 赛事配置表 (tournamentConfig)

| 字段名 | 类型 | 说明 |
|--------|------|------|
| TournamentID | 文本 | 主键，赛事唯一标识 |
| 赛事名称 | 文本 | 显示名称 |
| ChatID | 文本 | 关联的飞书群聊ID |
| BitableAppToken | 文本 | 该赛事使用的Bitable App Token |
| 赛事状态 | 单选 | 筹备中/进行中/已结束/已暂停 |
| 顺延开关 | 复选框 | 是否启用自动顺延 |
| 公示开关 | 复选框 | 是否启用卡组公示 |
| 首场签到开启方式 | 单选 | 手动/定时 |
| 默认签到超时分钟数 | 数字 | 默认超时时间 |
| 赛事备注 | 文本 | 备注信息 |

## 2. 战队主档案库 (teamMaster)

| 字段名 | 类型 | 说明 |
|--------|------|------|
| TeamID | 文本 | 主键，战队唯一标识 |
| 战队名称 | 文本 | 显示名称 |
| 队长OpenID | 文本 | 队长飞书OpenID |
| 队长姓名 | 文本 | 队长姓名 |
| 联系方式 | 文本 | 手机号或邮箱 |
| 战队状态 | 单选 | 正常/退赛/禁赛 |
| 注册时间 | 日期 | 报名时间 |
| 备注 | 文本 | 其他信息 |

## 3. 选手主档案库 (playerMaster)

| 字段名 | 类型 | 说明 |
|--------|------|------|
| PlayerID | 文本 | 主键，选手唯一标识 |
| 所属战队ID | 文本 | 关联TeamID |
| 游戏ID | 文本 | 炉石游戏ID |
| 选手姓名 | 文本 | 真实姓名 |
| 飞书OpenID | 文本 | 选手飞书OpenID |
| 注册时间 | 日期 | 报名时间 |
| 状态 | 单选 | 正常/退赛/禁赛 |

## 4. 赛事对阵与赛果表 (matchResults)

| 字段名 | 类型 | 说明 |
|--------|------|------|
| MatchID | 文本 | 主键，场次唯一标识 |
| TournamentID | 文本 | 关联赛事ID |
| 场次序号 | 数字 | 第几场 |
| 战队A | 文本 | 战队A的TeamID |
| 战队B | 文本 | 战队B的TeamID |
| 计划时间 | 日期 | 计划开始时间 |
| 实际开始时间 | 日期 | 实际开始时间 |
| 实际结束时间 | 日期 | 实际结束时间 |
| 比分 | 文本 | 如 "3:2" |
| 场次状态 | 单选 | 未开始/进行中/已结束/延期 |
| 签到状态 | 单选 | 未下发/已下发/已签到/超时 |
| 签到卡片消息ID | 文本 | 飞书消息ID |
| 备注 | 文本 | 其他信息 |

## 5. 卡组提交与审核表 (deckSubmission)

| 字段名 | 类型 | 说明 |
|--------|------|------|
| DeckID | 文本 | 主键，提交唯一标识 |
| TournamentID | 文本 | 关联赛事ID |
| 所属战队ID | 文本 | 提交战队ID |
| 选手ID | 文本 | 提交选手ID |
| 卡组代码 | 文本 | 炉石卡组代码 |
| 卡组截图 | 附件 | 截图文件 |
| 提交时间 | 日期 | 提交时间 |
| 审核状态 | 单选 | 待审核/已通过/已拒绝 |
| 审核人 | 人员 | 审核管理员 |
| 审核时间 | 日期 | 审核时间 |
| 审核备注 | 文本 | 拒绝原因等 |

## 6. 管理员白名单 (adminWhitelist)

| 字段名 | 类型 | 说明 |
|--------|------|------|
| OpenID | 文本 | 主键，管理员飞书OpenID |
| 姓名 | 文本 | 管理员姓名 |
| 权限级别 | 单选 | 超级管理员/赛事管理员/只读 |
| 可管理赛事 | 文本 | 可管理的TournamentID列表，逗号分隔 |
| 添加时间 | 日期 | 添加时间 |
| 添加人 | 文本 | 添加人OpenID |
| 备注 | 文本 | 其他信息 |

## 7. 审计日志表 (auditLog)

| 字段名 | 类型 | 说明 |
|--------|------|------|
| LogID | 文本 | 主键，日志唯一标识（自动） |
| 操作类型 | 文本 | 如"暂停顺延"/"手动签到"等 |
| 操作人OpenID | 人员 | 操作者 |
| 目标赛事ID | 文本 | 关联赛事ID |
| 目标场次 | 文本 | 关联场次序号 |
| 操作时间 | 日期 | 操作时间 |
| 操作详情 | 文本 | JSON格式详细数据 |
| 操作结果 | 单选 | 成功/失败 |
| 失败原因 | 文本 | 失败时的错误信息 |
