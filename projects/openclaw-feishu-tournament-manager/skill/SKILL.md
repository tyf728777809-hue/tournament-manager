# Tournament Manager Skill

## 概述

本 Skill 将赛事管理系统封装为 OpenClaw 可调用的能力，直接使用飞书插件工具操作 Bitable，无需额外配置飞书应用凭据。

## 核心能力

### 1. 赛事配置管理
- 读取赛事配置（使用 `feishu_bitable_app_table_record`）
- 更新赛事状态（顺延开关、公示开关等）

### 2. 战队与选手管理
- 查询战队注册信息
- 验证战队和选手存在性

### 3. 对阵与赛果管理
- 查询场次信息
- 更新场次状态
- 比分录入与校验

### 4. 签到流程
- 手动触发签到
- 自动顺延签到（基于比分）
- 签到卡片发送

### 5. 卡组管理
- 卡组提交审核
- 卡组公示

### 6. 审计日志
- 记录管理员操作
- 记录系统事件

## 数据表结构

依赖以下 7 张 Bitable 数据表：

1. **赛事配置表** (tournamentConfig) - 赛事基础配置
2. **战队主档案库** (teamMaster) - 战队信息
3. **选手主档案库** (playerMaster) - 选手信息
4. **赛事对阵与赛果表** (matchResults) - 场次与比分
5. **卡组提交与审核表** (deckSubmission) - 卡组数据
6. **管理员白名单** (adminWhitelist) - 管理员权限
7. **审计日志表** (auditLog) - 操作日志

## 使用方式

在 OpenClaw 会话中直接调用相关功能，Skill 会自动使用当前用户的飞书授权操作 Bitable。
