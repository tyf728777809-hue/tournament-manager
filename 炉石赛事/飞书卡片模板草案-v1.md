# 飞书卡片模板草案 v1

> 炉石赛事自动化系统 - 可复用卡片 JSON 骨架
> 版本: v0.1 | 日期: 2026-03-28

---

## 1. 模板约定

### 1.1 公共变量
- `{{title}}`：卡片标题
- `{{subtitle}}`：副标题/状态
- `{{record_id}}`：当前记录ID
- `{{tournament_uid}}`：赛事UID
- `{{entity_uid}}`：选手或战队UID
- `{{operator_open_id}}`：当前操作人

### 1.2 公共按钮回调
- `approve_*`
- `reject_*`
- `request_*`
- `view_*`
- `refresh_*`
