# 个人赛报名实现说明 v1

> 日期：2026-03-28

## 本轮已实现

已在 `callback-handler.js` 中新增个人赛报名处理：

- 新增 action：`submit_player_registration_form`
- 新增函数：`submitPlayerRegistrationForm(params, context)`
- 支持以当前提交人的 `operatorOpenId` 识别本人报名
- 支持按 `feishu_open_id` 查找并复用既有 `players` 资料
- 若不存在历史资料，则新建 `players` 记录
- 新建 `registrations` 赛事报名记录
- 自动发送报名成功回执通知

## 当前报名校验

- `real_name` 必填
- `player_id` 必填
- `nickname` 必填
- `bnet_id / battle_net_id` 必填
- 若赛事存在报名截止字段：
  - `registration_deadline`
  - `signup_deadline`
  - `apply_deadline`
  则会自动拦截超时报名
- 同一 `tournament_uid + player_uid` 只允许提交一次
