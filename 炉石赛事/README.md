## 🔄 文档层级关系

```
飞书设计稿（源头）
    │
    ├──→ 设计层文档（状态机、字段、卡片、映射）
    │       │
    │       └──→ 实现层代码（工具函数、回调框架）
    │               │
    │               └──→ 实战记录（测试数据、实例）
    │
    └──→ 实战直接验证（测试赛 HS202603-TEST-DSTL）
```

**原则：**
- 设计稿变更 → 同步更新设计层文档 → 再更新实现层
- 实现层发现的问题 → 反馈到设计层 → 再同步到设计稿
- 实战记录 → 验证设计层和实现层的正确性
